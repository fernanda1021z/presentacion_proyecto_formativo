Arquitectura de Software

Patrones de Diseño de Software

🖥️

Arquitectura Cliente-Servidor

El navegador (cliente) envía solicitudes HTTP a un servidor Apache que ejecuta la lógica en PHP y consulta la base de datos MySQL.

🧩

MVC (Modelo-Vista-Controlador)

El backend separa el Modelo (consultas a MySQL), la Vista (templates HTML/PHP) y el Controlador (recibe peticiones e invoca al modelo), facilitando mantenimiento y trabajo en equipo.

🔌

API REST

Endpoints organizados por recurso (/api/productos, /api/pedidos) que desacoplan frontend y backend y permiten integrar la pasarela de pagos y una futura app móvil.

📦

Patrón Repositorio

Encapsula todas las consultas a la base de datos en clases repositorio, evitando SQL disperso en los controladores y facilitando cambiar de motor de BD a futuro.

Buenas Prácticas

Estándares de Codificación

Reglas y convenciones que se seguirán durante el desarrollo para garantizar un código limpio, legible y fácil de mantener.

🔤

Nomenclatura

Tablas BD en snake_case plural (usuarios, productos), claves primarias id_{tabla}, variables PHP/JS en camelCase, clases PHP en PascalCase y constantes en UPPER_SNAKE_CASE.

🐘

Reglas de Codificación PHP

Indentación de 4 espacios, líneas máximo de 120 caracteres, uso obligatorio de PDO con consultas preparadas y separación en capas: model/, controller/, views/.

⚡

Reglas de Codificación JavaScript

Uso de ES6+ (const/let, arrow functions, async/await), sin uso de var, y módulos separados por responsabilidad: cart.js, checkout.js, admin.js.

🗄️

Reglas de Base de Datos

Toda tabla con clave primaria AUTO_INCREMENT id_{tabla}, fechas en DATETIME con DEFAULT NOW(), contraseñas siempre en hash bcrypt y transacciones para operaciones multi-tabla.

🔗

Rutas y Archivos

Rutas API en kebab-case plural (/api/productos), archivos PHP en snake_case, IDs HTML en kebab-case y clases CSS bajo la metodología BEM.

📝

Documentación del Código

Comentarios PHPDoc obligatorios en todas las funciones públicas y manejo de errores con try/catch en cada llamada asíncrona a la API.
